## About NicheMoat

> Stop guessing. Start ranking. Automated niche research on Etsy.

NicheMoat is a **Chrome extension** that runs in your browser using your existing eRank and Etsy sessions. One seed keyword in — winning product concepts out.

It runs a four-step pipeline:

1. **eRank research** — pull keyword data.
2. **Etsy snapshot** — capture live listings.
3. **Listing audit** — analyze the competition.
4. **Concept scoring** — verdicts: `home_run`, `worth_test`, or `skip`.

Plus real-time urgency signals (in-cart counts, 24-hour sales, view counts) and **one-click HTML reports** — like the ones shared in this repo.
